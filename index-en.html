Illustration of Common Cultural Framework Concept

As part of the R2.13 resource, students were asked to create a newspaper front page reporting facts about IUT Arles. We interpreted the objective of this work as designing a credible and engaging front page for a newspaper intended for IUT students ("L'Écho de l'IUT"). This work is based on knowledge of the concept of common cultural framework, namely a set of information shared by several parties, here IUT students. For the layout, we heavily relied on the plan present in the document distributed during the class where the instructions for this assignment were explained.

Qualitative Algorithm Comparison

As part of a cross-curricular end-of-semester project, we were asked to implement several different sorting algorithms and compare them, mainly on execution speed criteria. I subsequently wrote the synthesis above in English presenting the results and conclusions we could draw from them.

Technical aspects: Implementation of 5+ algorithms (QuickSort, MergeSort, BubbleSort, HeapSort), precise performance measurement with C++11 chrono, temporal and spatial complexity analysis (Big O), tests on different dataset sizes (10³ to 10⁶ elements).

Skills developed: Fundamental algorithms and complexity analysis, benchmarking methods and performance measurement, technical writing in English with scientific synthesis, and statistical analysis of results.

Route Calculator

This project was carried out as part of a first-year BUT Computer Science SAé. The objective was to use knowledge of graph theory, data management, and interface development using Qt in a single application meeting a clear need (calculating a route).

Technical aspects: Implementation of Dijkstra's algorithm for shortest path calculation, management of complex data structures (graphs, adjacency matrices), graphical interface with Qt Widgets and event management, data persistence system with CSV/XML files, manual memory management in C++.

Skills developed: Advanced algorithms and graph theory, object-oriented programming, user interface development with Qt, debugging and performance optimization, and technical documentation and unit testing.

Recipe Library

As part of the last project of the first year, we had to create a recipe manager using the Qt Creator tool. This C++ project allowed me to realize everything from A to Z, from designing classes to the structure of the graphical application to the datasets to use. My teammate having participated little, I realized that when things don't progress fast enough, you have to know how to take charge and manage on your own.

Skills developed: UML design and modeling, relational databases and SQL queries, design patterns (MVC, Singleton), individual project management and technical autonomy, and integration testing and functional validation.
