Route Calculator

This project was carried out as part of a first-year Computer Science Bachelor's project. The objective was to use knowledge on graph theory, data management as well as interface development using Qt in a single application responding to a clear need (Calculate a route).

Keyp - Password Manager

This Java project using JavaFX is a secure password manager developed by second-year students in the Computer Science BUT program. The application allows users to store and manage encrypted credentials with an expiration system and automatic alerts.

Technically, I implemented three interchangeable encryption algorithms (AES, DES, Blowfish) using the Strategy pattern, an MVC architecture with clear separation of responsibilities, and a persistence system with the DAO pattern. The JavaFX interface offers a dynamic table with color coding based on expiration urgency.

I was able to develop my skills in design patterns, applied cryptography (javax.crypto, SHA-256) and software architecture.
