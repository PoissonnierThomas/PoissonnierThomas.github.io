Calculateur de Trajet

Ce projet a été réalisé dans le cadre d'une SAé de première année de BUT Info. L'objectif était d'utiliser les connaissances sur la théorie des graphes, la gestion de données ainsi que le développement d'interface à l'aide de Qt dans une application unique répondant à un besoin clair (Calculer un trajet).

Ce logiciel a été réalisé en C++ et a nécessité pas mal de maintenance et de recherches afin d'obtenir un rendu final proche de ma conception initiale du projet. Tehniquement, j'ai implémenté l'algorithme de Dijkstra pour le calcul des plus courts chemins, géré des structures de données complexes (graphes, matrices d'adjacence) et des données dans des fichiers CSV XML.

Ainsi, j'ai pu développé mes compétences en algorithmique avancée, programmation orientée objet et tests unitaires.

Vous trouverez dans le fichier compressé de téléchargement le logiciel ainsi que la documentation qui va avec, notamment le README.

Keyp - Gestionnaire de mots de passe

Ce projet Java avec JavaFX est un gestionnaire de mots de passe sécurisé développé en deuxième année de BUT Informatique. L'application permet de stocker et gérer des identifiants chiffrés avec un système d'expiration et d'alertes automatiques.

Techniquement, j'ai implémenté trois algorithmes de chiffrement (AES, DES, Blowfish) interchangeables grâce au pattern Strategy, une architecture MVC avec séparation claire des responsabilités, et un système de persistance avec le pattern DAO. L'interface JavaFX propose un tableau dynamique avec code couleur selon l'urgence des expirations.

J'ai pu développer mes compétences en design patterns, cryptographie appliquée (javax.crypto, SHA-256) et architecture logicielle.

Vous trouverez ci-dessous le code source complet du projet avec sa documentation JavaDoc.
